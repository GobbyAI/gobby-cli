## Purpose

This file defines the JSON contract for the `gwiki` local-first wiki CLI. It declares the tool metadata and contract version, shared global flags like `--format` and `--quiet`, scope-selection options for choosing a project/topic, and a long command list where each command specifies its summary, whether it is daemon-consumed, its positional arguments and flags, and the JSON keys it returns. The `error_codes` section closes out the contract with the CLIâ€™s standardized failure responses.
